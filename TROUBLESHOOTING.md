# 이미지 추가 오류 해결 가이드

## 문제 해결 단계

### 1. 이미지 파일 확인
이미지 파일이 올바른 위치에 있는지 확인하세요:
```
ui/public/images/americano-ice.jpg
```

### 2. 브라우저 콘솔 확인
개발자 도구(F12)를 열고 Console 탭에서 에러 메시지를 확인하세요.

### 3. 네트워크 탭 확인
개발자 도구의 Network 탭에서 API 요청이 실패했는지 확인하세요.

## 일반적인 문제와 해결 방법

### 문제 1: "메뉴를 찾을 수 없습니다" 에러
- **원인**: 메뉴 ID가 올바르지 않음
- **해결**: 페이지를 새로고침하고 다시 시도

### 문제 2: "요청에 실패했습니다" 에러
- **원인**: 서버가 실행되지 않았거나 네트워크 문제
- **해결**: 
  1. 백엔드 서버가 실행 중인지 확인 (`npm run dev` in server folder)
  2. 브라우저 콘솔에서 정확한 에러 메시지 확인

### 문제 3: 이미지가 표시되지 않음
- **원인**: 이미지 파일이 없거나 경로가 잘못됨
- **해결**:
  1. `ui/public/images/` 폴더에 이미지 파일이 있는지 확인
  2. 파일명이 정확한지 확인 (대소문자 구분)
  3. 브라우저에서 `http://localhost:5173/images/americano-ice.jpg` 직접 접근 테스트

### 문제 4: 데이터베이스 업데이트 실패
- **원인**: 데이터베이스 연결 문제 또는 권한 문제
- **해결**:
  1. 데이터베이스가 실행 중인지 확인
  2. `.env` 파일의 데이터베이스 설정 확인
  3. 서버 로그 확인

## 디버깅 방법

### 1. 브라우저 콘솔 확인
```javascript
// 개발자 도구(F12) > Console 탭
// 에러 메시지와 스택 트레이스 확인
```

### 2. 네트워크 요청 확인
```javascript
// 개발자 도구(F12) > Network 탭
// PATCH /api/admin/menus/:id 요청 확인
// Status Code와 Response 확인
```

### 3. 서버 로그 확인
```bash
# 서버 터미널에서 로그 확인
# 에러 메시지와 스택 트레이스 확인
```

## 올바른 사용 방법

1. **이미지 파일 준비**
   - 파일명: `americano-ice.jpg`
   - 위치: `ui/public/images/americano-ice.jpg`

2. **관리자 화면에서 수정**
   - 아메리카노 메뉴의 수정 버튼(✎) 클릭
   - 이미지 파일명 입력: `americano-ice.jpg`
   - 수정 버튼 클릭

3. **확인**
   - 주문하기 화면에서 이미지가 표시되는지 확인
   - 브라우저 콘솔에서 에러가 없는지 확인
